BenchReadDecimal.hs

N.B., the naming convention for the candidate implementations changed significantly between version 0.3 and 0.4. The old benchmarks are retained primarily to show the differences between different architectures running 64-bit Linux, as well as between the same architecture running under different operating systems. Nothing changed in version 0.5, the results are just provided for reference.

BenchReadExponential.hs

These files only show (1) the old implementation from bytestring-lexing-0.4.3.3, (2) the new/current bytestring-lexing implementation, and (3) various competitor libraries (of which bytestring-read is the main competitor). To see the non-winning candidate implementations, uncomment them in the benchmarking file. This benchmark is the same as bytestring-read's benchmark.
